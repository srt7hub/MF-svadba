# Свадебное приглашение — Марат и Фирюза

Сайт-приглашение на свадьбу 12.09.2026 (Pravda Hall, Уфа).
Стек: Vite + React 19 + TypeScript + Tailwind CSS v4 + Motion.

## Запуск локально

```bash
npm install
npm run dev      # http://localhost:3000
```

## Сборка

```bash
npm run build    # результат в dist/
npm run preview  # локальный просмотр production-сборки
```

## Деплой на Vercel

Проект статический. На Vercel настройки определяются автоматически
(Framework: Vite, Build: `npm run build`, Output: `dist`).
Дополнительная конфигурация — в `vercel.json`.
